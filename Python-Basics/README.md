# CFTR Protein Basic Analysis (Python)

## Objective
To perform basic protein sequence analysis of the CFTR protein
using Python scripting.

## Tasks Performed
- Calculated protein length
- Counted amino acid composition
- Calculated GC content using Python string methods

## Skills Demonstrated
- Python scripting
- Biological data handling
- Result interpretation

## Industry Relevance
Such basic sequence QC steps are commonly used in
bioinformatics pipelines before advanced analysis.
