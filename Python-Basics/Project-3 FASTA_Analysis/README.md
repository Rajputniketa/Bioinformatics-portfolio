# Automated FASTA File Analysis (Python)

## Objective
To automatically read and analyze a protein FASTA file
using Python scripting.

## Tasks Performed
- Read FASTA file
- Extract protein sequence
- Calculated sequence length
- Counted nucleotide characters

## Industry Relevance
Automated FASTA parsing is a foundational step in
bioinformatics pipelines used in genomics, proteomics,
and variant analysis.
